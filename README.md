# README

The Server Status Checker

This programm uses:
* ruby 2.6.6
* rails 6.1.3
* sqlite3 1.4
* bootstrap 4.6
* yanr 1.22.10

To run the app it is needed to first:

bundle install

yarn add bootstrap@4.6

rails db:create db:migrate db:seed

After that, it is necessay to:

rails s

The app will be hosted in the Localhost:3000
